# dustinjaeger.net — IT Portfolio

A living, breathing portfolio site that gets better every night.

## What This Is
- **Domain:** dustinjaeger.net
- **Hosting:** Cloudflare Pages (free)
- **Source:** GitHub repo → auto-deploys to Cloudflare
- **Design:** Dark theme, teal accents, animated grid background

## How It Works
Every night at 2am, an automated build job (via Hermes Agent) improves this site:
- New features, sections, or visual enhancements
- Content updates and improvements
- SEO, performance, and infrastructure work
- Each night builds on what came before
